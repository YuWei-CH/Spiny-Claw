# Benchmarks

This directory stores benchmark outputs produced by Spiny-Claw runs.

Use it for:

- benchmark summaries
- correctness reports
- profiling outputs
- per-run logs
- leaderboards

Suggested structure:

```text
benchmarks/
└── <task-name>/
    ├── latest/
    ├── history/
    ├── profiles/
    └── leaderboard.json
```

See [`benchmarks/CONTRACT.md`](/home/yuwei/Documents/Spiny-Claw/benchmarks/CONTRACT.md) for the minimum output contract.
